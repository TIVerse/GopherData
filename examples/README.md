# GopherData Examples

This directory contains example programs demonstrating GopherData functionality.

## Phase 1 Examples

### basic_example.go

Demonstrates Phase 1 features:
- Creating DataFrames from maps
- Column selection
- Row filtering
- Series numeric operations (mean, sum, min, max)
- Null value handling (SetNull, FillNA, DropNA)
- CSV reading and writing

Run with:
```bash
cd examples
go run basic_example.go
```

## Expected Output

The example creates a DataFrame, performs various operations, and writes/reads CSV files.
